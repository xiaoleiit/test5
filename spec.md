# Project Specification

## What do you want to build?

创建一个最小 Java 命令行 Hello World 项目。使用单文件 Main.java 作为入口，不要使用 Maven、Gradle、Spring 或其他框架。程序运行后输出 Hello World。

## How do you consider the project is success?

1. 仓库根目录存在 Main.java
  2. 运行 javac Main.java && java Main 输出 Hello World
  3. 仓库根目录存在 README.md，写清编译和运行步骤
  4. 仓库根目录存在 .gitignore
  5. 所有文件提交到 Git 仓库
